# Knight Travails

A solution to the Knight Travails problem using Javascript.

## The Problem

Knight Travails is a problem in the domain of Chess. If given enough turns, a Knight piece can reach any square on a standard chess board in a maximum of 6 moves no matter it's starting position. With this problem, the goal is to create a program that takes inputs for an origin square, where the Knight will be placed, and a target square, where the Knight will end up. The program should output the shortest path it takes the Knight to travel from the origin to the target.
